# Phoebus Bob files

This bob files for Phoebus/Control System Studio (CSS) deployments at
SEES beamlines, especially at GSECARS.

These files have been auto-converted from ADL files, and are now ready
to be edited in place using CSS.  Several of these files have already
been edited to work better with Phoebus.

Please avoid overwriting files here with auto-converted files unless
you are sure the version here has not been edited.

These files can be edited with CSS.
